# hw_2_TP
1)Система для управления рецептами. (консольное приложение, которое позволяет создавать блюда, добавлять их в "рецепты", масштабировать порции и генерировать список покупок).
2)Описание проекта;
Проект включает следующие классы:
  1. Ingredient — хранит ингредиент с названием, количеством и единицей измерения
  2. Recipe — рецепт блюда с возможностью масштабирования
  3. ShoppingList — список покупок, который включает ингредиенты из одного или нескольких рецептов
  4. DietaryRecipe —  рецепт с указанием диетической категории (например, "веган", "без глютена")
3) Запуск
  1. Установите pytest:
  pip install pytest
  2.Запустите тесты в терминале:
  pytest
